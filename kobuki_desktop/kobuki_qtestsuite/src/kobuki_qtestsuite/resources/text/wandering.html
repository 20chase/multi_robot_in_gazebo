RQ108/RQ109 - Wandering/Bump Test

Overview

Put the robot in wandering mode and display the bump counter.

Usage - Bump Test

You do not need to start for this test.
  • Place hammering device in front of the bumper (left, centre or right)
  • Keep an eye on the bump counter

Usage - Safe Wandering

  • Place on a table
  • Configure speeds appropriately
  • Press start
  • It will use bump and cliff sensors to avoid a fall
