SP1 - Payload

Overview

Test the motion of the base under payload.

Usage

  • Start with a light payload.
  • Connfigure speed settings appropriately
  • Press start
  • It will map out a square
  • Repeat with a heavier payload and/or different surface, speed settings

Errata

This is quite inaccurate - it is just setting 'simple' goals relative to each side or turn's starting position and ignoring accumulated errors.
